Projects

FIM comparison tool

The FIM comparison tool is a work in progress whose goal is to enable the rapid evaluation of rasters of flood maps. To learn more and find links to the existing codebase visit the FIM comparison tool project page.

ChannelWatch

ChannelWatch is a planned future Water Prediction Node project that will use USGS 3DEP and other lidar data to help with channel geometry and floodplain characterization in the National Water Model. The ChannelWatch project page contains an outline of the proposed approach.
